6. Proceso después PCB: Montaje Final Sistema Medición de Temperatura

Las siguientes imágenes relatan el proceso experimentado después del diseño de la PCB. Se puede observar la primera PCB, la cual no funcionó debido a problemas de diseño: múltiples puntos se tocaban simultáneamente, causando una incorrecta salida del amplificador.

Para esto, se fabricó otra PCB con un diseño más estandarizado y mayor separación entre caminos. Paralelamente, se trabajó en pruebas de depuración como el circuito de acondicionamiento de la señal del puente de Wheatstone, utilizando un LM358N para ajustar el voltaje de la señal al A0 del Arduino.

Además, se diseñaron y soldaron circuitos fuente para obtener 6V de offset y 5V de alimentación. Finalmente, se probó el circuito completo junto con el controlador, logrando mostrar la temperatura y cumplir con el propósito del sistema.
